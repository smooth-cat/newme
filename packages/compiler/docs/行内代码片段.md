支持行内声明模板片段 并向子组件传递

行内模板赋值时需要包含以下属性，才能在子组件成功渲染

```ts
export class InlineFragment {
  /** 提供代码的 tokenizer */
  public tokenizer: Tokenizer  
  /** 快照 记录行内模板的起始位置 */
  public snapshot: Partial<Tokenizer>,
  /** 通过 getData 获取的数据 */
  public data: any,
  /** 快照赋值给子组件的键 */
  public key: string,
}
```

## API

```typescript
class Parent {
	ui = bobe`
		${Child} header=
      div text="header头"
    | 
      div	text="children"
	`
}


class Child {
	ui = bobe`
		div
			div id="header"
				{header}
		div
    	div id="children"
    		{children}
	`
}
```



Compiler 解析支持

对于默认 children 尝试复用之前逻辑，可能需要类型提示

对于其他属性，在 = 和 \n 后调用 parseXXXNode 来继续解析内容
